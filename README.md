# Tax Calculator (SCD Lab Project)

## Project Description

This is a Java Swing-based Desktop Application designed to calculate tax amounts efficiently. It provides a user-friendly interface for inputting income data and computes the tax based on defined slabs and rules, ensuring accuracy and ease of use.

## Requirements Implemented

1. Graphical User Interface (GUI): Built using Java Swing (JFrame, JTextField, JLabel, etc.) for an interactive experience.
2. Event Handling: Calculates tax instantly upon clicking the "Calculate" button using ActionListeners.
3. Exception Handling: Robust validation to handle invalid inputs (e.g., negative numbers, empty fields, or non-numeric characters).
4. Code Refactoring (Modular Design): Logic is separated from the UI into a dedicated class to ensure clean and maintainable code.
5. Unit Testing: Automated test cases written using JUnit 4 to verify tax calculation logic and edge cases.

## Setup Instructions

1. Open Eclipse IDE and create a new Java Project.
2. Import the source files from the src folder.
3. Add JUnit 4 to your project's Build Path.
4. Run the main GUI class as a Java Application.
5. Run the JUnit test class to verify the calculations.
